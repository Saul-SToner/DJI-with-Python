## Current Focus

This repository is not a claim of a completed optical design. The current organized branch focuses on:

1. Zemax/ZOS-API workflow archiving.
2. Wide-angle candidate structure search.
3. Ray-trace and vignetting verification.
4. Failure-surface and margin analysis.
5. Data preparation for future machine-learning-based initial-structure screening.

## Important Notes

- Raw `.ZOS` and `.ZMX` design files are not tracked in Git.
- Generated result folders and local binary outputs should remain excluded.
- Optical conclusions should only be stated when supported by existing logs or fresh-load verification.
- Unknown or unverified values should remain marked as `UNKNOWN` or `TODO`.
